# GOSS - Grand Operational Simulation Series Assistant

A web-based procedural play assistant for the Grand Operational Simulation Series (GOSS) board wargames by Decision Games.

Currently supports: **Wacht am Rhein** (Battle of the Bulge)

## Features

- **Sequence of Play Stepper** — walk through each phase and sub-phase of a game turn
- **Rules Reference** — searchable rules organized by section
- **Turn Tracking** — tracks current game turn, date, and time of day
- **Progress Persistence** — saves your place via localStorage
- **iPad-friendly** — responsive design, installable as a PWA

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS
- PWA-ready
